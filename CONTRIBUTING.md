# Contributing

Clone, install, copy the env file:

```bash
git clone https://github.com/gamersindo1223/MAL-ReadmeList
cd MAL-ReadmeList
npm install
cp .env.example .env  # fill in your credentials
```

Run it locally with `node index.mjs`, run tests with `npm test`.

If you're adding a new feature, add a test for it. If you're fixing a bug, make sure the existing tests still pass. Open a PR and describe what you changed and why — that's it.

Report bugs as GitHub Issues. Include your MAL username or a minimal example that reproduces the problem.
